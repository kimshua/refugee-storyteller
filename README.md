# Refugee Storyteller

An interactive web experience that brings to life the journeys of refugees who have found their way to South Korea. Through immersive storytelling, ambient soundscapes, and cinematic design, each story invites you to walk in someone else's shoes.

## Stories

- **North Korea** - The Shadow of the 38th Parallel
- **Afghanistan** - The Thin Line
- **Myanmar** - The Spring Revolution's Exile
- **Ukraine** - Fields of Sunflowers and Ash
- **Syria** - The Jasmine Left Behind

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Router

## Getting Started

**Prerequisites:** Node.js (v18+)

1. Clone the repository:
   ```bash
   git clone https://github.com/kimshua/refugee-storyteller.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
```
